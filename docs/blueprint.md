# **App Name**: StudyTrack Journal

## Core Features:

- Activity Logging: Log all student activities throughout the day, including study sessions, classes, and breaks.
- Problem Tracking: Track the number of questions solved per day and categorize them by subject or type. User can select a category and insert a short text response.
- Progress Visualization: Visually display progress using charts and graphs, showing activity levels and problem-solving trends over time.
- Intelligent Scheduling Assistant: An AI tool which looks at activity history to recommend future study times, optimizing for productivity and balancing activities to avoid burnout.
- Data Persistence: Persistently store user journal entries and historical activity data.

## Style Guidelines:

- Primary color: A calm blue (#64B5F6) to promote focus and concentration.
- Background color: Light gray (#F5F5F5) for a clean and unobtrusive workspace.
- Accent color: A vibrant orange (#FFAB40) for highlights and important CTAs, promoting sense of energy.
- Body and headline font: 'PT Sans', a humanist sans-serif for readability and modern look
- Use clean, minimalist icons for activity types and navigation.
- A clean, intuitive layout with clear sections for logging activities, tracking problems, and viewing progress.
- Subtle animations and transitions to enhance user experience.
